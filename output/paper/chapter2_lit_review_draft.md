# 第二章 文献综述与理论基础（Phase 1 初稿）

## 2.1 经典股利理论：从“无关”到“机制竞争”
围绕“公司为何分红”这一核心问题，经典研究形成了若干互补但并不完全一致的解释框架。早期研究强调在完美市场条件下股利政策的中性命题，而后续文献逐步引入信息不对称、代理冲突与投资者偏好等现实摩擦，从而使分红行为成为公司治理、融资约束和市场定价的综合结果。

在信号理论视角下，分红被视为管理层向外部投资者传递企业质量或现金流可持续性的手段（R03）。在代理理论框架下，现金分红可减少管理层可支配自由现金流，降低过度投资和资源挪用空间（R01、R02）。迎合理论则进一步指出，分红决策会受投资者阶段性偏好影响，企业可能通过调整分红来匹配市场“股利溢价”（R04）。

总体上，经典理论提供了“为什么分红”的机制集合，但在中国制度背景下，各机制的相对权重并非固定，需要结合产权结构、监管强度与资本市场发展阶段做经验识别。

## 2.2 公司治理、代理成本与生命周期：分红动因的结构化证据
现有实证研究显示，分红行为不是单一财务指标驱动，而是公司治理质量、生命周期位置与外部监督强度共同作用的结果。机构投资者持股、大股东掏空风险、内部治理约束等变量均与分红决策存在显著关联（R05、R06、R16、R17）。

与此一致，近期研究开始从“异质性公司行为”而非“平均效应”视角识别分红动因。生命周期文献表明，不同成长阶段企业在现金流稳定性、再投资需求和融资成本上系统不同，从而导致分红政策的阶段性差异（R22）。激励契约与管理者行为偏差也会影响分红选择（R21）。

在中国上市公司情境下，陈运森等（2024）使用机器学习框架对分红驱动因素进行预测式识别，显示生命周期与代理成本变量具有突出的解释力（R19）。该结果推动文献从“单一回归显著性比较”转向“预测性能 + 变量重要性 + 可解释性”的综合评估范式。

## 2.3 监管政策与市场反应：从“半强制”到“硬约束”
中国现金分红监管经历了从窗口指导、半强制约束到强化硬约束的演进过程。既有研究围绕半强制政策已积累较多证据，显示政策能够在一定程度上提升分红概率和分红水平，但政策效果受到公司治理、再融资动机与外部制度环境影响（R12、R13、R14、R15）。

新一轮“硬约束”政策研究进一步关注监管压力差异下的处理效应。卿小权等（2025）基于 2020-2024 年沪深 A 股主板样本的 DID 证据表明，政策对原本低分红公司有显著提升效应，且在法治环境较弱、非国有、代理问题更突出的样本中更强（R20）。这意味着政策并非“平均作用于所有公司”，而是通过约束边际更高的企业实现治理改进。

此外，2023-2025 年国际文献也在扩展分红政策的外生冲击来源，包括制度质量、数字金融、环境约束与行为偏差等维度（R21 及近三年补充文献）。该趋势说明分红研究正从单一公司财务问题，扩展为“治理-监管-市场环境”联动的综合议题。

## 2.4 方法论进展：ML 识别与 DID 因果评估的互补性
在研究方法上，机器学习与准自然实验并非替代关系，而是识别“相关性结构”和“因果效应”的互补工具。随机森林和梯度提升树可在高维变量场景下提升预测性能并捕捉非线性关系（R07、R08），结合 ALE 等解释工具可增强经济含义解读能力（R09）。

另一方面，DID 方法可在政策冲击背景下识别平均处理效应及异质性效应，但在多期处理与错位实施时需注意估计偏误并采用更新的识别框架（R10、R11）。据此，将“第3章 ML 动因识别”与“第4章 DID 政策评估”联立，可以构建从预测证据到因果证据的闭环：先识别关键动因，再检验政策是否改变这些动因主导下的分红行为。

## 2.5 文献评述：共识、分歧与可改进点
### 2.5.1 共识
- 分红行为受代理冲突、信息不对称与生命周期特征共同影响，而非单因素决定。
- 监管政策对分红具有可观影响，但政策效应显著依赖公司特征与制度环境。
- 方法上，单一线性模型难以完整刻画分红决策，需引入非线性预测与因果识别并行框架。

### 2.5.2 分歧
- 对“分红提升是否一定改善公司价值”的结论并不一致，不同市场阶段和融资约束下可能出现方向差异。
- 对“监管政策到底是纠偏还是扭曲”的判断存在争议：部分研究强调治理改进，部分研究关注企业策略性应对。
- 对“生命周期变量与治理变量谁更关键”的证据在样本、模型与时期上并不稳定。

### 2.5.3 可改进点与本文动机
- 多数文献将“分红动因识别”和“政策因果评估”分离处理，缺少统一研究框架。
- 既有政策评估对异质性机制刻画仍有限，尤其缺乏与 ML 识别结果的联动验证。
- 新监管环境下（硬约束强化）需要更近期证据，且需同时回答“哪些企业会分红”与“政策如何改变分红行为”两个问题。

基于此，本文构建“ML 预测 + DID 评估”的双路径设计：第3章识别分红关键动因，第4章检验监管冲击的因果效应与异质性机制，从而增强研究结论的可解释性与政策含义。

## 2.6 理论框架与研究假设（衔接）
结合前述文献，本文提出三项可检验假设：
- H1（预测筛选假设）：生命周期变量与代理成本变量在分红预测中的重要性显著高于其他变量。
- H2（政策因果效应假设）：硬约束政策显著提高处理组公司的分红意愿与分红水平。
- H3（异质性处理效应假设）：政策效应在成熟期企业与代理问题更突出的企业中更强。

对应变量、模型设定与预期符号详见 `notes/hypothesis_mapping.md`。
