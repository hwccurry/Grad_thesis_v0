# 第三章 变量定义表

更新时间：2026-02-27

## 一、样本说明

- **样本范围**：沪深A股上市公司，2006—2022年（17个年度截面）
- **样本量**：31,469 家-年观测值
- **数据来源**：CSMAR 数据库，经参考文献（陈运森等，2024）整理
- **训练范式**：一年期滚动预测（以 t 年样本训练，t+1 年样本测试），共 16 轮（2006→2007, 2007→2008, …, 2021→2022）
- **标准化**：以 2006 年截面拟合 StandardScaler，统一变换所有年度

## 二、被解释变量

| 变量名 | 中文名 | 度量方式 | 类型 |
|--------|--------|----------|------|
| Dividend_ratio1 | 股利支付率 | 每股现金股利 / 每股收益（主回归响应变量） | 连续 |
| Dividend_ratio2 | 股利收益率 | 每股现金股利 / 每股股价 | 连续 |
| Dividend_ratio3 | 股利/资产比 | 现金股利总额 / 总资产 | 连续 |
| Dividend | 是否发放现金股利 | 当年发放现金股利=1，否则=0（分类任务响应变量） | 二元 |

## 三、解释变量（35个连续/离散特征 + 36个行业虚拟变量）

### 3.1 公司治理与代理成本类

| 变量名 | 中文名 | 度量方式 | 预期方向 |
|--------|--------|----------|----------|
| Managefee_ratio | 管理费用率 | 管理费用 / 营业收入 | 负 |
| Manageshare | 管理层持股比例 | 管理层持股数 / 总股数（%） | 正 |
| Indep_ratio | 独立董事比例 | 独立董事人数 / 董事会总人数 | 正 |
| Bgender | 董事会女性比例 | 女性董事人数 / 董事会总人数 | 不确定 |
| Bshare | 董事长持股比例 | 董事长直接持股 / 总股数 | 不确定 |
| Bage | 董事长年龄 | ln(董事长年龄) | 不确定 |
| Btenure | 董事长任期 | ln(任职月数) | 不确定 |
| Bsalary | 董事长薪酬 | ln(年度薪酬总额) | 不确定 |
| Equity | 股权激励虚拟变量 | 当年实施股权激励=1，否则=0 | 正 |
| Da_abs | 财务报告质量 | 修正Jones模型残差绝对值（操纵性应计） | 负 |
| Tunneling | 其他应收款资产比 | 其他应收款 / 总资产（掏空代理变量） | 负 |
| Top1 | 股权集中度 | 第一大股东持股比例（%） | 不确定 |
| Sharebalance | 股权制衡度 | 第2—5大股东持股和 / 第1大股东持股 | 正 |
| Minorityrate | 中小股东持股比例 | 非前十大股东持股比例（%） | 正 |
| Institution | 机构投资者持股比例 | 机构投资者合计持股（%） | 正 |
| Pledge | 控股股东股权质押比例 | 控股股东质押股数 / 持股数（%） | 负 |

### 3.2 生命周期与盈利能力类

| 变量名 | 中文名 | 度量方式 | 预期方向 |
|--------|--------|----------|----------|
| Retainedearn_ratio | 留存收益资产比 | 留存收益 / 总资产（生命周期代理变量） | 正 |
| ROA | 资产收益率 | 净利润 / 总资产 | 正 |
| Cashflow | 每股经营活动现金流量 | 经营活动净现金流 / 总股数 | 正 |
| Freecash2 | 自由现金流 | 自由现金流量 / 总资产 | 正 |
| Growth | 销售增长率 | (本期营业收入 - 上期) / 上期营业收入 | 负 |
| Dividend_lag | 上一期股利水平 | 上年度股利支付率 | 正 |

### 3.3 税收与融资约束类

| 变量名 | 中文名 | 度量方式 | 预期方向 |
|--------|--------|----------|----------|
| Tax_avoid | 税收规避程度 | 法定税率与实际税率之差 | 不确定 |
| Tax_ratio | 实际税率 | 所得税费用 / 利润总额 | 不确定 |
| Tax_volatility | 纳税波动率 | 近三年实际税率标准差 | 不确定 |
| Constraint | 融资约束程度 | KZ 指数（数值越大约束越强） | 负 |
| Refinance | 再融资动机 | 当年有再融资行为=1，否则=0 | 正 |

### 3.4 市场环境与估值类

| 变量名 | 中文名 | 度量方式 | 预期方向 |

| Soe | 产权性质 | 国有控股=1，否则=0 | 不确定 |
