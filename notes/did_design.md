# DID 准自然实验设计（Phase 3 / Task 3.1）

更新时间：2026-02-28

---

## 1. 政策背景

2023年12月，中国证监会发布修订后的《上市公司现金分红指引》（以下简称"2023指引"），对上市公司现金分红提出更明确的量化标准和约束机制。该政策的核心条款包括：

- **量化门槛**：要求上市公司最近三年累计现金分红金额不低于年均净利润的30%，或累计分红总额不低于5000万元。
- **"不分红即解释"机制**：对不符合分红条件的公司实施更严格的信息披露要求。
- **差异化监管**：根据公司盈利状况和现金流特征制定差异化的分红标准。

该政策修订构成了对中国上市公司分红行为的外生监管冲击，为运用双重差分方法评估政策效应提供了准自然实验条件。

## 2. 处理组与对照组划分

### 2.1 划分逻辑

**处理组（treat=1）**：政策实施前（2020—2022年）现金分红未达监管门槛的公司。具体标准为以下任一条件：
- 2020—2022年年均现金分红金额低于年均净利润的30%；
- 累计现金分红低于5000万元。

**对照组（treat=0）**：政策实施前已满足分红门槛的公司。该类公司在政策实施前已具有较高分红水平，受政策的边际约束较弱。

### 2.2 样本结构

| 分组 | 观测数 | 占比 |
|------|--------|------|
| 处理组（treat=1） | 4,879 | 54.2% |
| 对照组（treat=0） | 4,125 | 45.8% |
| 合计 | 9,004 | 100% |

### 2.3 外生性论证

处理组的划分依据是公司在**政策实施前**（2020—2022年）的历史分红行为，而非政策实施后的时变特征。这一设计的外生性体现在：
1. **基于历史行为的分组**：处理变量取决于政策实施前三年的分红水平，公司无法在政策发布前预判并策略性调整分组归属。
2. **政策的非选择性**：2023指引对所有A股上市公司同时生效，不存在自选择进入处理组的通道。
3. **时间不可逆性**：政策时点（2023年）在样本期中明确固定，不随公司行为变化。

## 3. 样本与时间窗口

- **样本期**：2020—2024年（5年），覆盖政策前3年与政策后2年。
- **政策时点**：2023年（post=1 for 2023—2024）。
- **横截面**：2,306家沪深A股上市公司（剔除金融行业与ST公司后）。
- **观测值**：9,004个公司-年度观测。

| 2022 | 1,829 | 976 | 853 |

## 4. 模型设定

### 4.1 基准 DID 模型

$$Y_{it} = \alpha_i + \lambda_t + \beta \cdot did_{it} + \gamma X_{it} + \varepsilon_{it}$$

其中：
- $Y_{it}$：被解释变量（DivDummy 或 DivPayRate）
- $\alpha_i$：公司固定效应
- $\lambda_t$：年份固定效应
- $did_{it} = treat_i \times post_t$：核心解释变量
- $X_{it}$：控制变量向量
- $\varepsilon_{it}$：误差项，标准误聚类到公司层面

### 4.2 被解释变量

| 变量 | 定义 | 类型 | 均值 |
|------|------|------|------|
| DivDummy | 当年是否发放现金股利（是=1） | 二元 | 0.914 |
| DivPayRate | 现金股利支付率（每股现金股利/每股收益） | 连续 | 0.420 |

### 4.3 控制变量

| 变量 | 定义 |
|------|------|
| SIZE | 公司规模（总资产的自然对数） |
| AGE | 上市年龄（ln(上市年限+1)） |
| LEV | 资产负债率 |
| ROA | 资产收益率 |
| GROWTH | 营业收入增长率 |
| CFO | 经营现金流/总资产 |
| TOP | 前五大股东持股集中度 |
| INDEP | 独立董事比例（%） |
| MH | 管理层持股比例（%） |
| HHI_BANK | 地区银行业赫芬达尔指数 |
| MKT | 地区市场化程度指数 |

## 5. 识别策略与有效性检验

| 检验 | 方法 | 目的 |
|------|------|------|
| 平行趋势 | 事件研究法（动态DID） | 验证政策前处理组与对照组趋势平行 |
| 安慰剂检验 | 随机分配政策时间（1000次） | 排除结果由随机因素驱动 |
| PSM-DID | 倾向得分匹配后重新估计 | 缓解处理组与对照组的协变量不平衡 |
| 熵平衡 | 熵平衡加权后重新估计 | 另一种协变量平衡方法 |
| Hausman-Taylor | Hausman-Taylor估计 | 处理时不变内生性 |
| 排除替代解释 | did×Capital_AR, did×Asset_GR交互项 | 排除资本留存与资产增长的干扰 |
| 剔除再融资 | 排除refinance=1的样本 | 排除再融资动机对分红的干扰 |

## 6. 异质性分析维度

| 分组变量 | 定义 | 与假设的对应 |
|----------|------|------------|
| SOE | 产权性质（国有=1） | 国有企业合规驱动更强 → 政策效应可能更弱 |
| agc_high | 高代理成本（=1） | H3：代理问题突出的企业政策效应更强 |
| insinv_high | 高机构持股（=1） | 外部监督强的企业可能政策边际效应更弱 |
| law_high | 高法治水平地区（=1） | 法治环境好的地区政策执行力更强 |
