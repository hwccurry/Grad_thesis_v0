## 一、风险与待办逐项说明

### 风险 1：DivDummy 的 pre_3 系数在 5% 水平显著

**现象**

平行趋势检验中，DivDummy 的 t-3 期（2020年）系数 = 0.0402，t 值 = 2.127，p 值 = 0.033。这意味着在常规 5% 水平下该系数统计显著，可能被解读为"平行趋势假设不成立"。

**为什么这不是致命问题**

1. **参考文献（卿小权）使用的是 99% 置信区间**。其 Stata 代码 `coefplot, levels(99)` 明确以 99% CI 为判断标准。在 99% CI 下，pre_3 的置信区间为 [-0.009, 0.089]，包含零 → 不拒绝平行趋势。
2. **pre_2（t-2期）完全不显著**（系数 0.0224，t=1.249，p=0.212），且离政策时点更近，更具诊断力。
3. **DivPayRate 的 pre_3 和 pre_2 均不显著**（p=0.11, p=0.13），第二个被解释变量完全满足平行趋势。
4. **系数量级对比**：pre_3 = 0.0402 远小于政策后系数（t=0: 0.1077，t+1: 0.0965），即使存在微弱的前趋势，政策实施后的跳升仍然显著且明确。

**应对策略**

- 正文已按 99% CI 标准报告平行趋势成立（与参考文献一致）。
- 如答辩或外审质疑，可补充两个辅助论证：(a) 缩短事件窗口（去掉 t-3，只保留 t-2 作为前期检验）；(b) 在附录中报告 95% CI 的事件研究图并承认 pre_3 的边际显著性。

**结论**：可辩护，无须修改基准结论，但应在答辩中有准备。

### 风险 2：H3 仅"部分支持"

**现象**

四个异质性维度中，只有**代理成本（agc_high）**的组间差异在 10% 水平上统计显著：

| 维度 | DivDummy 组间差异 | 显著性 | DivPayRate 组间差异 | 显著性 |
|------|------------------|--------|-------------------|--------|
| 代理成本 | +0.0529 | * (p=0.059) | +0.0644 | * (p=0.085) |
| 产权性质 | -0.0350 | (p=0.138) | -0.0230 | (p=0.470) |
| 机构持股 | -0.0257 | (p=0.458) | -0.0262 | (p=0.523) |
| 法治水平 | -0.0228 | (p=0.361) | -0.0400 | (p=0.248) |

**为什么这是合理的**

1. **代理成本维度是 H3 的核心预测**。写作框架中的假设 H3 为"政策效应在代理问题更突出的企业中更显著"——这一点恰好得到支持。
2. **其余三个维度方向一致**：非国有 > 国有、低机构持股 > 高机构持股、低法治 > 高法治——方向均符合经济直觉（政策约束在监督更弱的企业中边际效应更大），只是未达到统计显著。
3. **组间差异检验本身统计功效较低**：Wald 检验（或 bootstrap差异检验）要求两组系数差异除以合并标准误后 |z| > 1.96，这对样本量和效应量的要求远高于单组回归。
4. **参考文献也使用 bdiff 命令做 bootstrap 差异检验**，预期其部分维度的组间差异同样不显著——这是该类研究的常见结果。

**应对策略**

- 正文已措辞为"**假设 H3 部分得到支持**"，并分维度解释了经济含义。
- 答辩中可强调：核心维度（代理成本）显著是最重要的；其余维度提供了方向性证据。

**结论**：合理且诚实，无须强行修改结论。

### 2.2 linearmodels vs. reghdfe 的等价性说明

Python `linearmodels.PanelOLS` 与 Stata `reghdfe` 均采用**组内变换（within transformation）**消除固定效应，并支持公司层面聚类标准误。两者的数值差异来自：

1. **浮点精度**：Python 使用 64 位浮点运算，Stata 默认使用 double（也是 64 位），精度等价。
2. **单例处理**：参考文献使用 `keepsingletons` 保留单例观测，我们的 PanelOLS 默认也保留单例——行为一致。
3. **自由度校正**：两者在聚类标准误的自由度校正方式上可能存在微小差异（影响 t 值的小数第三位以后），不改变显著性判断。

**结论**：使用 Python 复现 Stata 的 reghdfe 在方法论上完全合理，不构成方法缺陷。

### 2.4 安慰剂检验的迭代次数差异

参考文献使用 1000 次模拟，我们使用 500 次。这一差异的影响如下：

- 500 次已足够刻画安慰剂系数的分布形态（均值约为 0，呈对称正态）。
- 真实 DID 系数（0.0828 / 0.1081）远离分布尾部（> 3σ 以外），即使增加到 1000 次也不会改变结论。
- 如需完全对齐参考文献，可将脚本中 `N_PLACEBO = 500` 改为 `1000`。

### 2.5 异质性检验方法差异

| 方面 | 参考文献 | 我们 |
|------|---------|------|
| 分组方式 | 按二元变量分组回归 | 相同 |
| 组间差异检验 | Stata `bdiff` (bootstrap, 1000 reps) | Wald z 检验 ($z = \frac{\beta_1 - \beta_0}{\sqrt{SE_1^2 + SE_0^2}}$) |
| 统计功效 | bootstrap 可能更精确 | Wald 是大样本近似，在 N>4000 时与 bootstrap 基本一致 |

二者的差异在大样本下可忽略。如需完全对齐，可考虑用 Python 实现 bootstrap 差异检验（计算量较大但可行）。
